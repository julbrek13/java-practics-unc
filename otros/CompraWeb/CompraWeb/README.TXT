Examen 19/2/2013

1) Implemente el metodo "validarCompra" de la clase TiendaProxy
   de acuerdo a la documentación del método (1 método)
 
2) Implemente la clase "Paginador" tal que implemente la Interfaz
  "IPaginador".  (4 métodos)
   Modifique el constructor de la clase "Catalogo" para inicializar
   la variable "IPaginador paginador"
    
3) Implemente la clase "CarritoDeCompras" tal que implemente la
   Interfaz "ICarrito" (5 métodos)

4) Implemente la clase "FiltroPorCategoria", tal que implemente la
   clase "IFiltro". Este filtro devuelve un lista de Items filtrados
   cuya Categoria coincida con el criterio de filtrado. (3 métodos)
   El criterio de filtrado (String criterio) debe convertirse al tipo
   Categoria (utilice Categoria.valueOf() para converir el criterio)
   
5) Implemente la clase "FiltroPrecioMenor", tal que implemente la
   clase "IFiltro".
   Este filtro devuelve un lista de Items cuyo precio sea menor al
   criterio de filtrado. (3 métodos)
   El criterio de filtrado (String criterio) debe convertirse al
   tipo Double.
   
Puede utilizar la clase Cliente para probar su implementación de las
diferentes interfaces/clases solicitadas.
